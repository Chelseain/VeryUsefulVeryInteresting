今日世界，已臻新元。网络互通，人机互联。数据杂多，信息驳繁。纵千般甄别，小有所获，终其萤火，微薄甚焉。故余生宏愿，与诸君共勉：集万人之力揽尽妙网，以效始皇，收天下之兵，聚之咸阳！
